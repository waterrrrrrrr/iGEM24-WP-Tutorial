Overview

The software engineering is mainly carried out from the following two directions: On the medical worker side, a web app “BioProtoType” is developed to verify the usability of Telepathy kit; On the patient side, an Android/iOS downloadable software for collecting hardware medical data is designed.

On the medical worker side

BioProtoType is a versatile statistical computing tool that integrates methods such as data visualization and hypothesis testing. The software utilizes clinical statistical methods to provide accurate and insightful visualizations to validate the usability of the wet laboratory product Telepathy Kit. At the same time, the software retains more room for future development (such as linked database, K8S deployment, sequencing, etc.).

logo of web app

Available Function

Confussion Matrix: validate the utility of Telepathy by compare its diagnosis with known result and compute common statistical values(Sensitivity & Specificity).

Kaplan-Meier Method: predict the survival timeline of those using telepathy and traditional diagnosis method, compare and discover the advantages of telepathy over traditional diagnostic methods

Log-Rank Test: integrated in the module 'Survival Prediction' with Kaplan-Meier Method, act as the patch of Kaplan-Meier Method in the situation of similar survival timeline

Key Features

Cache. If same function meets the same data, it will output the result directly to speed calculation.

Developed by the framework "Streamlit". Streamlit is an open-source Python library that makes it easy to create and share beautiful, custom web apps for machine learning and data science.

Deployed by Docker. Docker is an open source application container engine based on the Go language and open source Apache2.0 protocol.Docker allows developers to package their applications and dependencies into a lightweight, portable container that can then be distributed to any popular Linux machine, as well as virtualization. Containers are completely sandboxed, there will be no interface between each other (similar to iPhone apps), and more importantly, the container performance overhead is extremely low.

On the patient side

In order to cooperate with the work of Hardware and search for data sources for BioProtoType, we designed an Iot-based software "Futuria", which is connected to the Telepathy kit via Bluetooth, receives the data transmitted by the detection box in real time and performs algorithm processing, and finally presents the visual disease analysis results. To provide a basis for further medical help, doctors can also conduct relevant clinical analysis based on the data collected in the cloud.

Quick to Download

Android/iOS converts the apk into a QR code for others to scan and download. The QR code will only appear in physical form with our hardware to prevent potential cyberattack risks. More info in https://fir.im/

User-friendly UI design

Considering that middle-aged and elderly people are the main audience of the kit, our software adopts barrier-free design in the UI design

Privacy concern

To solve the privacy problems, we designed a system based on the combination of full-homomorphic encryption algorithm and AES algorithm to realize the storage, management, retrieval and calculation of medical data related to the kit.

The data retrieval calculation method based on full-homomorphic encryption technology can directly perform basic calculation on the encrypted data without decrypting it into plain text, which not only ensures the security of user data, but also ensures that the server can directly operate the stored data ciphertext, balancing the relationship between ensuring the security of user data in the cloud storage system and serving cloud computing. The combination of AES encryption algorithm also effectively makes up for the low encryption rate of all homomorphic encryption algorithm
